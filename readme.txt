Easily set a custom text for your "read more" links.

== Description ==

Who isn't tired of all those standard "read more..." links on the blogs? Well, with this simple plugin your can customize your "read more" link for every single post.
This does also do you some good for your SEO, because anchor text does matter :)

== Installation ==

1. Install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
